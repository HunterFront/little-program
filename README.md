1、原生js实现打地鼠游戏  

http://hunterfront.github.io/little-program/beatmole  

参考原型图：  

![](https://github.com/HunterFront/little-program/blob/master/images/game1.png)  

相关技术：HTML、CSS、JS  

项目需求：实现一个web页面的打地鼠游戏  

功能需求：  

	随机在界面中生成地鼠  

	击中地鼠的同时实现一定的交互效果（地鼠、锤子状态有变化）  

	记录分数、关卡，并计算命中率  

	实现游戏的开始、暂停和重新开始功能  

	游戏结束时，显示成绩
